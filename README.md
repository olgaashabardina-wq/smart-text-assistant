# Smart Text Assistant

Консольный AI-ассистент с переключением режимов:

- Normal Mode — стандартная модель
- Thinking Mode — reasoning модель

## Возможности

- Переключение моделей при запуске
- Работа через OpenRouter (OpenAI-compatible API)
- Сохранение истории диалога
- Обработка ошибок и таймаутов
- Подсчёт usage / reasoning tokens

## Технологии

- Python
- OpenAI SDK
- OpenRouter API
- Prompt Engineering

## Запуск

pip install -r requirements.txt  
python text_agent.py
